1. Upload scan

For best reliability, run ceiling-rcp-init <folder> from a terminal and open the printed URL — that bypasses Safari's file-picker quirks. The web upload below is a fallback.

2. Trace polygons

  1. Click vertices, then click the first one (or press Enter) to close.
  2. Defines the height datum — its average height becomes 0 mm.
  3. Recesses → +mm above main; bulkheads → −mm below main.

Replaces main + regions with up to 5 height-clustered planes covering 100% of the room outline. Run after drawing the room.

What the colour shading means: inside each polygon, every pixel is tinted by how far its scanned height sits above (lighter) or below (darker) that polygon's own average. A polygon that looks evenly tinted is a flat plane; obvious light/dark patches inside one polygon mean it has clipped a bulkhead or recess — shrink the polygon or split it.

Tools

Click a vertex to drag.

3. Tidy borders

Pushes / pulls every polygon so they share clean borders inside the room — no gaps, no overlap. Main ceiling fills any space regions don't claim.

4. Export

Coloured masks, relative-height tags on each polygon, and dimensioned room outline.
